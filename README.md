# MyDIffDrivePlugin  
This is a sample program for my class.  

## How to use.  
Download this repository from the github.  

    $ git clone https://github.com/m-shimizu/MyDIffDrivePlugin  

Build the program.  

    $ cd MyDIffDrivePlugin  
    $ mkdir build  
    $ cd build  
    $ cmake ..  
    $ make
